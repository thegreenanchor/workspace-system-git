# Social Campaign Output Standard

Source of truth: internal Notion standard (sanitized in this export)

Use this standard for any deliverable that outputs a social media campaign page, campaign calendar, or week-by-week post package intended for Notion, Buffer, and Adobe Firefly workflows.

## Required output shape

Produce a Notion campaign page using this exact section order:

1. `Campaign Overview`
2. `How to use this page`
3. `Quick Schedule`
4. `Firefly Notes`
5. `Buffer Notes`
6. `Week-by-Week Content`
7. `Quick Reference`

## Weekly post structure

For every post, include:
- `Theme`
- `Alt hook A`
- `Alt hook B`

Then include these labeled code blocks:
- `Buffer caption`
- `Overlay title`
- `Overlay subtitle`
- `Firefly prompt`

## Defaults and guardrails
- Default cadence: Monday, Wednesday, Friday unless changed
- No invented facts, stats, or testimonials
- No unconfirmed pay claims or guarantees
- No logos, patient faces, or readable private info in image prompts
- Keep copy specific, grounded, and paste-ready
